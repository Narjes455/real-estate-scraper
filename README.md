# 🤖 أتمتة فتح Google Chrome باستخدام بايثون

أول مشروع لي في أتمتة المتصفحات باستخدام **Python** و **Selenium**.  
يقوم الروبوت بفتح متصفح Chrome وزيارة موقع Google تلقائياً.

---

## 🛠️ التقنيات المستخدمة

- **Python 3.14+**
- **Selenium** – للتحكم في المتصفح
- **WebDriver Manager** – لإدارة سائق Chrome تلقائياً

---

## 📁 هيكل المشروع

```
real-estate-scraper/
├── scraper.py          # الكود الأساسي
├── requirements.txt    # المكتبات المطلوبة
└── README.md           # شرح المشروع
```

---

## ▶️ طريقة التشغيل خطوة بخطوة

1. تأكدي من تثبيت **Python** على جهازك.
2. حمّلي المكتبات المطلوبة:

```bash
pip install -r requirements.txt
```

3. شغّلي الكود:

```bash
python scraper.py
```

---

## 📌 ماذا يفعل الكود؟

- يفتح متصفح **Google Chrome** تلقائياً.
- يتجه إلى موقع **Google**.
- يطبع رسالة نجاح في الـ Terminal.
